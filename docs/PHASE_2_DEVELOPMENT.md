# Fase 2 — Desenvolvimento local

O frontend desta fase será validado localmente antes de qualquer configuração de servidor.

## Escopo inicial

- gestão completa da consulta;
- edição de data, hora, profissional, notas e estado;
- visualização do histórico de alterações;
- tratamento consistente de erros da API;
- build de produção e testes de contrato em desenvolvimento.

## Endpoints locais

- Frontend: `http://localhost:5173`
- API: `http://localhost:8081`

Nenhum deploy de produção faz parte desta fase.
